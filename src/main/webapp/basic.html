CSS 문법

선택자 {속성:속성값; }

CSS 적용방식 3가지

  • External CSS - 외부파일방식. 스타일소스를 웹문서와 별개로 만들어 연결하는 형태로 적용.
    통일된 스타일 가능, 수정보수 용이.
  • Internal CSS - 내장방식. 스타일소스를 웹문서의 head 안에 만들어 적용
    웹문서에 종속 됨. (해당 페이지에만 적용)
  • Inline CSS - 인라인방식. 스타일소스를 태그 뒤에 직접 적용.
    소스가 길어지고 유지보수에 불리하고 통일성 있는 사이트 구축이 어려우나 css적용 우선순위가 가장 높음 작업환경이 본문만 수정가능할 때 사용.

대표 선택자 3가지

  • tag선택자 - 태그 그대로를 선택자로 활용. 자동적용, 강제적용.
  • id선택자 - id를 선택자(#id)로 활용. 중복적용 불가. 주로 구조적 목적에 쓰임.
  • class선택자 - class를 선택자(.class)로 활용. 중복적용 가능, 조합 가능. 주로 디자인 목적에 쓰임
